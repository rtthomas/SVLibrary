To catalog a book, first select the catalog, then enter a book number. The default value shown is just the next one in sequence after the value of the highest number in the catalog. (If you are cataloging a book which is already in the manual system, you may want to retain the original number.)
{{numberError}}
You can now upload the cover images, then generate a catalog page pdf file. This isn't necessary if the the catalog page already exists for this book.

If you don't have the cover images now, you can upload them and generate the pdf later on the "Edit a Catalog" tab. If you select and upload the wrong image, you can just repeat the operation before downloading the pdf.

Note: If you've uploaded an image, there is a short delay before it it is available for generating the PDF document. If the page is incomplete, just wait a few seconds, then click the Download Catalog Page button again.

Note: There is a short delay between when the images are uploaded and when they can be used to generate the page. If the page is incomplete, just wait half a minute or so, then click the Download button again.
